# 中英文翻译数据集（17245条）

#### 1、数据集介绍

这是一份结构清晰、内容实用的中英文短句对照数据集，收录了17245条短句数据，每条数据均由英文原句和对应中文译文组成，格式统一，便于系统接入与快速使用！
